## 🚀 How to Open in Obsidian

This repository is designed to be used as an **Obsidian Vault**. To open it:

1. **Clone or Download** this repository to your local machine.
2. Open **Obsidian**.
3. Click on the **Open folder as vault** button (if you have other vaults open, click the vault icon in the bottom left, then **Open...** next to **Open folder as vault**).
4. Navigate to and select the root folder of this repository.
5. You're all set! All the Markdown notes will be available in your side panel.
